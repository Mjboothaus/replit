Application Features

  • Enter GPS coordinates for any NSW coastal location
  • View current tide status and predicted tide times
  • See 48-hour tide forecasts with interactive charts
  • Explore historical tidal data for previously queried locations
  • Analyze tidal statistics across different areas

About This Application:

The NSW Tidal Information app provides real-time and predictive tidal data for coastal locations in New South Wales, Australia. It uses accurate location validation to ensure coordinates are within NSW coastal regions, and generates reliable tidal data for informed decision-making by boaters, fishers, and coastal visitors.
